API Observatory is a developer tool designed to help developers observe API usage patterns inside Chrome DevTools.

## Data Collection

API Observatory collects **only technical request metadata**:

- HTTP method (GET, POST, etc.)
- URL pathname (no query parameters required)
- HTTP status code
- Request duration
- Timestamp

## What We Do NOT Collect

API Observatory does **not** collect or store:

- Request bodies
- Response bodies
- HTTP headers
- Cookies
- Authentication tokens
- Personal or user-identifiable data

## Data Storage

- All data is processed **locally inside the browser**
- No data is transmitted to external servers by default
- Exported data (JSON/CSV) is initiated explicitly by the user

## Permissions Justification

- `<all_urls>`: Required to observe API requests on developer-selected pages
- `storage`: Used to store local preferences and license activation state
